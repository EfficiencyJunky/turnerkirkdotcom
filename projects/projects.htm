1) The FrankenPipe

A Bagpipe MIDI Controller with RC Car Control

I started this project in January of 2006 as an independent study at the University of Miami. Little did I know that it would quickly turn into an obsession of mine and one of my major passions in life. It won 1st Prize in The Student Design Competition at the 121st Audio Engineering Society Convention.

2) Super Duper Destroytion!!!

A MatLab GUI with 6 Digital Distortions using a Sine Wave and Guitar Sample as inputs

I embarked on this project with my friend and classroom colleague Tim Yamaya. We set out to create digital distortions that would show the full gamut of digital distortions from a complete hard clipping of the signal to a very "analog" sounding rounded off soft clipping distortion.

You can view the original horribly uninformed concept webpage here, or download the GUI Directly below

The Destroytion Webpage

The Destroytion MatLab GUI

To access the GUI:
  1. extract the zip file to a directory
  2. Open up MatLab and set the "Current Directory" to be the same as the Destroytion directory.
  3. Type dgui in the MatLab "Command Window" and voila! Super Duper Destroytion is yours!
